FlyTrap Contact Form Example

The Flytrap contact form allows for the ability to place contacts subscribed into a FlyTrap Contact list of your choice. These contact list can be used to send updates, announcments, and other types of communications within your funnel pages.

Start Developing 

    <script>
        //Initialize Components
        fly.init({
            appId: "eafe707a0f4508d6941e66fa912e1140",
            customUrl: "",
            textColor: "#454545",
            showImages: true,
        }).then(function (res) {
            console.log("Successfully Loaded:", res);
            if (res) {
    /*The "FlyTrap Demos is the Contact list that will be generated if it doesn't exist.  You can change his"*/
                fly.LoadForm("flytrapcontact", "FlyTrap Demos");
            }

        }); 
        </script>
