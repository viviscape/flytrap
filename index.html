Flytrap JS SDK Examples

Getting Started

The FlyTrap JS SDK is pretty simple to use. Listed below are the base CDN References to get started within your header section.

FLYTRAPCDNJS
        
        <head>
        <link href="https://cdn.ftgo.io/v1/fly-form.css" rel="stylesheet"/>
        <script src="https://cdn.ftgo.io/v1/fly.min.js"></script>
        
        ...
        </head>

FlyTrap Contact Form Example

The Flytrap contact form allows for the ability to place contacts subscribed into a FlyTrap Contact list of your choice. These contact list can be used to send updates, announcments, and other types of communications within your funnel pages.
